#🎓 Academic Performance Tracker

A dynamic web application that calculates semester-wise GPA and credit-weighted CGPA using JavaScript, with persistent data storage and a responsive UI design.

#🚀 Features

📚 Semester-wise GPA calculation (Sem 1 – 8)

🎯 Credit-weighted CGPA calculation (University-accurate formula)

💾 Persistent data storage using localStorage

🔄 Dynamic semester switching

🧮 Grade-to-point conversion system

🛡️ Input validation & defensive error handling

📱 Responsive and clean UI design

#🏗️ Tech Stack

HTML5 – Structure

CSS3 – Styling & layout

JavaScript (ES6) – Logic & dynamic functionality

localStorage API – Data persistence

#📂 Project Structure
├── index.html
├── style.css
├── script.js
└── README.md

⚙️ How It Works

Select grades for each subject.

Click Calculate GPA for the semester.

GPA is automatically stored in browser storage.

Navigate to the CGPA tab.

Click Calculate CGPA to compute overall performance.

🛡️ Edge Case Handling

Prevents CGPA calculation without minimum semesters.

Avoids division by zero.

Handles missing GPA values safely.

Ensures numerical parsing before calculations.
